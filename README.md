# MY NOTES App

# Overview

MY NOTES is a user-friendly note-taking application built with React. It allows users to create, read, update, and delete notes seamlessly. The app is designed to provide a clean and efficient interface for managing notes.

# Features

	•	Create Notes: Add new notes with ease.
	•	View Notes: View all your notes in one place.
	•	Edit Notes: Update the content of your notes.
	•	Delete Notes: Remove notes that are no longer needed.
	•	Responsive Design: Works on both desktop and mobile devices.
 
# Installation
To set up the project locally, follow these steps.
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: npm i
   
 Running the App : npm run dev
